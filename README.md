🐍 Snake Game

A classic Snake Game built using JavaScript, HTML, and CSS.

🎮 Features

Smooth snake movement
Keyboard controls (Arrow keys)
Score tracking
Game over on collision (wall or self)
Simple and clean UI

🚀 How to Run

1. Clone the repository: https://github.com/Sakshi-khare/Snake_Game_Project
2. Open the project folder
3. Run the game:
4. Open index.html in your browser

🎯 Controls

⬆️ Up Arrow – Move Up
⬇️ Down Arrow – Move Down
⬅️ Left Arrow – Move Left
➡️ Right Arrow – Move Right

🛠️ Tech Stack

HTML
CSS
Javascript
